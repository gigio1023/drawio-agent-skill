# drawio-agent-skill

Create editable draw.io diagrams instead of one-off diagram output.

`drawio-diagram` is for cases where the `.drawio` file itself matters: architecture diagrams, system maps, editorial figures, and compact explainer visuals.

## Install

Preferred:

```bash
npx skills add gigio1023/drawio-agent-skill@drawio-diagram
```

### Codex

Tell Codex:

```text
Fetch and follow instructions from https://raw.githubusercontent.com/gigio1023/drawio-agent-skill/refs/heads/main/.codex/INSTALL.md
```

Detailed docs: `docs/README.codex.md`

## Use it for

- editable `.drawio` deliverables
- architecture diagrams
- routed flowcharts
- compact research or editorial figures
- OpenAI / Anthropic / Vercel style explainer layouts

## It emphasizes

- native draw.io first
- one page grammar at a time
- obvious component-to-component arrows
- editability over decorative XML

## What's inside

- `SKILL.md`
- references for grammars, layout safety, and visual patterns
- sample `.drawio` assets
- reference refresh script and archived reference data
